# 🏦 GenAI Financial Investigation Assistant

An AI-powered banking fraud investigation application built using **Python, Streamlit, Ollama, and Llama 3.2**.

## 🚀 Features

- 🤖 AI-powered fraud investigation
- 📂 Upload transaction datasets
- 📊 Interactive dashboard
- 🥧 Risk distribution pie chart
- 📈 Transaction amount analysis
- 🔍 Search transactions
- 🚨 Filter by risk level
- 📄 PDF report generation
- 📥 CSV report download
- 🔒 Local LLM using Ollama (Llama 3.2)

## 🛠 Tech Stack

- Python
- Streamlit
- Ollama
- Llama 3.2
- Pandas
- Plotly
- ReportLab
